# Notion Templates

This directory contains templates for the Control Tower Notion workspace.

## Databases

### Decisions Log
Track every strategic decision with rationale.

### Open Questions  
Backlog of unresolved strategic questions.

### Roadmap
What's planned, in progress, and shipped.

### Meeting Notes
Async "board meeting" summaries.
